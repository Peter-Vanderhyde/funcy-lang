# funcy-lang README

This extension provides syntax highlighting for the Funcy programming language. Funcy is meant to be a mix between the languages Python and C++.

## Features

The syntax highlighting includes:
- Keywords
- Strings
- Comments
- Block Comments
- Numbers
- Variables
- Functions
